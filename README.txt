RETINA CHECK - Eye Image Classification Web Application

============== What the project does ==============

This web-based image classification application allows users to upload images and receive a prediction on their category based on a pre-trained model.

============== Why the project is useful ==============

The application serves as a convenient tool for those wanting to swiftly classify images without the hassle of setting up or training their own model. Whether for educational, research, or development purposes, it offers a ready-to-use solution for eye image classification.

============== How users can get started with the project ==============

Installation and Running

1. Ensure Python is installed on your machine.
2. Install the required packages: pip install -r requirements.txt
3. Run the application: python application.py
4. Open a browser and navigate to `http://127.0.0.1:5000/`
